Vue Weather

Web.

Lors de mon parcours de spécialité “Développement web et mobile” au semestre 4, j’ai voulu en apprendre davantage sur les frameworks javascript dans le but de créer ma première application web.

Mon choix s'est porté sur Vue.js car, mis en comparaison avec React et Angular, il est le plus facile des trois à appréhender et celui-ci permettait selon moi d’aborder les frameworks javascript le plus aisément.

Vue Weather

Réflexion :

L’idée de présenter ce petit projet sous la forme d’une application météo m'est rapidement venu à l’esprit. Il me fallait pour cela une API d'analyse d’informations météorologiques, de laquelle je pouvais extraire ses données pour ensuite les manipuler et les afficher. Cela me permettait de différencier ce projet d’un simple exercice pratique mais également d’aboutir à un résultat concret, ce qui était pour moi une source de motivation non négligeable.

Développement :

Pour réaliser cette application météo j’ai donc fait appel à Open Weather Map, un service en ligne qui fournit des données météorologiques à échelle mondiale.

Le processus de récupération des données JSON se fait de manière asynchrone par requêtes HTTP en utilisant l'API Modern Javascript Fetch, ainsi que les méthodes et les conditions de Vue.js. Les données sont alors récupérées en temps réel via des appels à l'API rest de Open Weather Map qui renvoient les données météorologiques actuelles.
